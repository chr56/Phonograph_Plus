<b>Phonograph를 기반으로 독자적으로 개발 중인 버전</b><br />

Phonograph는 머티리얼 디자인이 적용된 가벼운 안드로이드용 로컬 음악 플레이어입니다.
Phonograph Plus는 더 이상 유지보수되지 않는 Phonograph의 <i>독립</i> 포크 버전으로,
2021년부터 유지보수 및 개발이 진행되고 있습니다. <br />

Phonograph Plus는 Phonograph의 모든 기능을 계승합니다:


- 경량
- 오프라인, 광고 없음, 개인정보 보호
- 클래식 머티리얼 디자인 2
- 자동 전환을 지원하는 나이트 모드 (라이트, 다크, 블랙)
- Monet을 지원하는 사용자 지정 테마 색상 (기본, 강조)
- 기본적인 오디오 재생 기능 (재생 속도 조절, 취침 타이머, 시스템 제공 외부 이퀄라이저 포함)
- 유연한 대기열 관리 (셔플, 반복, 드래그를 통한 자유로운 재정렬, 대기열 스냅샷)
- 가사 지원 (시간 동기화 .lrc 가사 및 텍스트 가사, 내장 또는 외부 파일 모두 지원)
- 높은 수준의 사용자 설정이 가능한 유연한 재생 알림
- 유연한 음악 라이브러리 구성: 곡, 아티스트, 앨범, 장르, 폴더(계층 없이 또는 트리 구조)별 정리, 다양한 순서로 정렬,
  다중 선택 지원, 다양한 레이아웃 스타일의 그리드 또는 목록 뷰
- 높은 수준의 사용자 설정이 가능한 유연한 메인 플레이어 UI
- Android Auto 지원
- 클릭 동작에 대한 높은 수준의 사용자 설정
- 경로 필터 (제외 모드 또는 포함 모드)
- 검색 지원
- 즐겨찾는 곡 지원
- 파일 플레이리스트 지원 (보기 및 기본 편집, 고정 기능 포함)
- 내부 데이터베이스 플레이리스트 지원
- 재생 기록 및 빈도 기록, 유연한 시간 간격별 최근 추가 곡 표시
- 향상된 오디오 메타데이터 뷰어 및 태그 편집기
- 앱 바로가기 및 위젯
- 다양한 앨범 아트 소스
- 설정 및 데이터 백업 지원
- 앱 내 언어 변경
- 등등...


더 자세한 정보는 GitHub 프로젝트 홈페이지를 방문하세요.

Phonograph Plus는 GitHub Releases(안정 버전 및 미리보기 채널)와 F-Droid(안정 버전 채널)에서 다운로드할 수 있습니다.

## **v1.8.7.LTS 2025.05.16**

This is a LTS release for branch 1.8.x.
此版本为 LTS 分支版本.

### EN

- This is a LTS release, an update after [v1.8.6.LTS](https://github.com/chr56/Phonograph_Plus/releases/tag/v1.8.6.LTS).
- This contains many vital fixes back-ported from _v1.9.3 ~ v1.10.1_.
- This LTS branch is going to be _inactive_, due to the difficulty of maintenance.


1. Fix: Sleep timer possibly not working on Android 14 and above (backported from v1.10.1)
2. Fix: randomly change artwork occasionally when toggling shuffle mode (backported from v1.10.0)
3. Fix: some abnormal behaviors when changing items display settings especially on Artist Page (backported from v1.9.3)
4. Improve: performance of image loading for albums and artists (backported from v1.9.3)
5. Translation: update Hungarian (by Balázs Úr & Balázs Meskó)
6. Development: update dependencies, upgrade Kotlin to 2.1.20, upgrade AGP to 8.9.2


### ZH

- 此版本为 LTS 版本, [v1.8.6.LTS](https://github.com/chr56/Phonograph_Plus/releases/tag/v1.8.6.LTS) 的更新
- 此版本包含从 _v1.9.3 ~ v1.10.1_ 版本移植而来的关键修复
- 由于维护困难，该 LTS 分支将不再活跃

1. 修复 睡眠定时器在 Android 14 及以上版本上可能失效 (移植自 v1.10.1)
2. 修复 切换随机播放模式时偶尔会随机更改封面 (移植自 v1.10.0)
3. 修复 更改项目显示设置特别是艺术家页面上时出现一些异常行为 (移植自 v1.9.3)
4. 改进 专辑和艺术家的图片加载速度 (移植自 v1.9.3)
5. 翻译 更新 匈牙利语 (Balázs Úr & Balázs Meskó)
6. 开发 更新依赖, 更新 Kotlin 至 2.1.20, 更新 AGP 至 8.9.2

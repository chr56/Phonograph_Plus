# **Additional Features or Enhancements Compared to Original Phonograph**

Here is an incomplete list compared to original Phonograph:

### User Interface

- Automatic dark mode that aligns with system settings.
- Pro unlock feature, supporting all available themes.
- <del>Poor</del> support of Monet.
- Redesigned File tab with a fresh look.
- Redesigned App Intro with a modern design.
- Enhanced user experience on "Library" pages.

... and various other user interface changes.

### Improvements

- Option to manually change the application language.
- Display Time Axis in "Lyrics" dialog and enable seeking based on the songâ€™s time axis, with support for lyrics following.
- Support more sort orders.
- Path filtering with the ability to exclude (blacklist) or include (whitelist) specific folders.
- Song menu shortcuts for adding items to the exclusion list.
- Enhanced notification image quality for Android T and later.
- Improved storage management experience for Android 11 and later.
- Option to delete song files along with corresponding external lyrics files.
- Increased capacity for played track history and top tracks (100->150).
- Enhanced multi-select functionality.
- Enhanced playlist support.
- Improved menu experience, by collapsing infrequently used options.
- Allow tabs fixed.

### Enhancements

- Playback speed control.
- Customizable click actions for songs.
- History tracking for the playback queue.
- Enhanced Main Player.
- Internal Database Playlist
- Newly designed Detail page with additional metadata such as tags.
- Updated Tag Editor with support for various tags.
- Android Auto compatibility.
- Ability to resume playback after interruptions.
- Option to collect app crash reports.
- Collaboration with the [StatusBar Lyric Xposed Module (API)](https://github.com/Block-Network/StatusBarLyric).
- Ability to export internal databases and settings for backup purposes.

... and many other minor features and fixes.

For a comprehensive list of changes, improvements, and new features, please refer to the [Changelog](https://phonographplus.github.io/changelogs/changeslogs/changelog.html).
